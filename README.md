
# Frends LLM Fine-Tuning

A project for fine-tuning Large Language Models to learn and understand the Frends BPMN process language.

## Overview

This project uses [Unsloth](https://github.com/unslothai/unsloth) for efficient fine-tuning of LLMs on Frends BPMN process definitions and documentation.

## Features

- Fine-tune LLMs on Frends BPMN syntax
- Efficient training with Unsloth
- Learn process automation patterns
- Generate Frends workflows

## Requirements

- Python 3.8+
- PyTorch
- Unsloth
- Frends documentation/samples
